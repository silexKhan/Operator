# 📝 MARKDOWN_ARCHITECT: The Document Architect

## 🧩 The Equation (문서 방정식)
**MD/PPTX ➔ [Text Extraction] ➔ [Logic Validator] = Doc Integrity**

- 마 문서는 단순한 기록이 아닌 시스템의 설계도(Blueprint)이다.
- PPTX조차 텍스트로 변환되어 구조적 검증의 대상이 된다.

## 🛡️ TDD Fence (TDD 울타리)
1. **[PRD Integrity]** 5대 필수 섹션(목적, 배경, 기능, 예외, 히스토리)이 존재하는가?
2. **[Terminology]** '유저'가 아닌 '사용자'로 용어가 통일되었는가?
3. **[Protocol 0-1]** '중략' 없이 모든 명세가 리터럴로 작성되었는가?
4. **[Format]** 날짜 형식이 `YYYY-MM-DD`를 준수하는가?
